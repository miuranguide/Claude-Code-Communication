# Broadcaster App

TikTok LIVE 配信用カメラ＋オーバーレイ＋BGMアプリ。

## 機能一覧

- カメラプレビュー（フロント/バックカメラ切替対応）
- 動画クリップオーバーレイ再生（位置・サイズ・回転・不透明度・音量調整）
- BGM再生（just_audio、ループ/ワンショット切替）
- WebSocketサーバー（Remote接続、QRコードペアリング）
- キュー機能（再生中のコマンドをキューイングして順番再生）
- カスタム音量プリセット（追加・編集・削除）
- クールダウン（連打防止）

## 必要権限

| 権限 | 用途 |
|------|------|
| カメラ | LIVE配信のカメラプレビュー |
| マイク | カメラプレビュー用（オプション） |
| ストレージ | 動画クリップ・BGMファイルの保存 |
| ネットワーク | WebSocketサーバー（Remote接続） |

## ビルド手順

```bash
cd apps/broadcaster
flutter pub get
flutter run
```

## 画面構成

- **ホーム画面**: カメラプレビュー + オーバーレイ + QR + STOP
- **素材管理**: 動画クリップ・BGMの追加/削除/ループ切替
- **レイアウト編集**: ドラッグ＆ピンチでオーバーレイ配置（回転スライダー付き）
- **設定**: クールダウン、キューモード、音量プリセット、サーバー情報
